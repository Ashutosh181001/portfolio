# Hi, I'm Ashutosh

I'm an applied AI engineer — I build agentic and GenAI systems on top of LLMs: orchestration, RAG, evals, guardrails, cost tracking, the whole loop. What fascinates me most about agentic AI is how creative and genuinely uncertain it can be. The interesting challenge, to me, is figuring out how to interpret and guide that creativity — to harness it in a controlled way so the uncertainty becomes something predictable and useful, rather than something to just work around.

My path here was a bit of a loop: started in ML, learned at my first placement that the hard part usually lives upstream of the model (in the data and the platform), went deep on data engineering and platform software, and then closed the loop by building an AI agent that automates that platform's own modelling work.

This repo is where the *learning* lives — separate from client work, no confidentiality constraints, just me figuring things out in the open.

## What this repo is about

I'm exploring agentic AI from first principles, and I wanted somewhere to actually build things rather than collect tutorials. So this is a running series of real, working projects — not toy demos — starting from "what is an LLM and how does it actually predict the next token" and working up to "deploy a properly productionised agent to the cloud," the way it'd actually be done on a real team.

Each project gets its own folder with its own README explaining what it explores and why. I'll add to this table as things land:
